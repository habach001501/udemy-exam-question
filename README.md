# CloudPro Simulator

A comprehensive exam simulation and review platform for AWS Certifications.

## Features

- **Exam Mode**: Timed simulation with random questions.
- **Review Mode**: Browse questions by topic with immediate feedback and explanations.
- **Results Tracking**: Visual breakdown of your performance and history tracking.
- **Responsive Design**: Fully responsive interface built with Tailwind CSS.
- **Dark Mode**: Optimized for comfortable reading.

## Tech Stack

- **Framework**: React 18 + Vite
- **Styling**: Tailwind CSS / FontAwesome
- **Routing**: React Router v6
- **State**: React Context (QuizContext)

## Getting Started

### Prerequisites

- Node.js (v16+)
- npm (v7+)

### Installation

1. Clone the repository
   ```sh
   git clone [repository-url]
   ```
2. Install dependencies
   ```sh
   npm install
   ```
3. Start the development server
   ```sh
   npm run dev
   ```

## Build

To build the project for production:

```sh
npm run build
```
